# QR Code Generator 🚀

A sleek and efficient command-line tool to generate high-quality QR codes from any URL or text input. Built with Node.js, it offers a seamless way to create and save QR images instantly.

## ✨ Features
- **Interactive CLI**: Simple and intuitive prompt-based interface.
- **Instant Generation**: Creates QR codes in seconds.
- **Auto-Save**: Saves QR codes as PNG images and logs the source URL to a text file.
- **Lightweight**: Minimal dependencies for fast execution.

## 🛠️ Built With
- [Node.js](https://nodejs.org/) - Runtime Environment
- [Inquirer.js](https://github.com/SBoudrias/Inquirer.js) - Interactive Command Line Interface
- [qr-image](https://github.com/alexeyten/qr-image) - QR Code Generation Library

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/anilgorade8-star/QR-Code-Generator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd QR-Code-Generator
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Usage
Run the generator with the following command:
```bash
node index.js
```
Follow the prompt to enter your URL, and your QR code will be generated as `qr_img.png`!

## 📂 Project Structure
- `index.js`: Main logic for the QR generator.
- `qr_img.png`: The generated QR code image.
- `URL.txt`: A text file containing the last generated URL.
